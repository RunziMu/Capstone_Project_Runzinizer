# Capstone Project: Runzinizer
This is the capstone project I made as the completion of Full Stack Web Devlopment Bootcamp

This web app is called **Runzinizer** as an combination of my firstname plus Organizer :)

Runzinizer is a web app that keep tracks of user's expenses and income, like a mini version of expense manager.
Runzinizer contains six responsive tabs, each tab contributes its own function.

**Home Tab**
This tab shows user's transaction history. Users are also able to view all transactions add more expense and income to transactions.
<img width="1260" alt="Home Page" src="https://github.com/RunziMu/Capstone_Project_Runzinizer/assets/101481214/34d35892-9eab-4eea-aa39-3fbe4fda9dbc">

**Search Tab**
This tab filters and displays all transaction from a sepecific category selected by user.
<img width="1261" alt="Search Page" src="https://github.com/RunziMu/Capstone_Project_Runzinizer/assets/101481214/a272c280-a471-44e2-94f2-6dea8733bca3">

**Report Tab**
This tab display Total income, total expense and total balance, plus the portion of each category. It also provides useful insights and recommendations to user regaring to their expense habit.
<img width="1261" alt="Report Page" src="https://github.com/RunziMu/Capstone_Project_Runzinizer/assets/101481214/cff017fa-de7e-45db-965c-2c6d3869c110">

**Register Tab**
Users will be asked to register a new account with Runzinizer if it's their first time use.
<img width="1185" alt="Register Page" src="https://github.com/RunziMu/Capstone_Project_Runzinizer/assets/101481214/03c57054-ff00-4a0e-b81f-76bacb43b783">

**Login Tab**
After registration, users are asked to login to Runzinizer, otherwise the **Profile** **Home** **Search** and **Report** will be unaccessible.
<img width="1184" alt="Login Page" src="https://github.com/RunziMu/Capstone_Project_Runzinizer/assets/101481214/ec344fc2-3920-48e9-9db2-3f9052ad179d">

**Profile Tab**
This tab shows user's basic information including name, email and *profile picture.
*Profile picture will show if the user set up during registration.
